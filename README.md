The trainer functions as an external tool. It reads the memory addresses assigned to your game instance. To keep your experience smooth, keep these tips in mind:

*   Save your game before you apply changes. This acts as a backup if you decide to revert your stats.

*   Close the trainer before you close the game. This provides a clean exit for both programs.

## ❓ Troubleshooting Common Issues

If the trainer does not function as expected, refer to this list of common solutions.

### The trainer does not detect the game
Check if the game is actually running. The trainer looks for active game processes. If you just opened the game, wait ten seconds and click the Refresh button inside the trainer window. Ensure that you run the game and the trainer with standard user permissions.

### Settings do not update
Verify that you have arrived at the start of a chapter or a gameplay screen. Some trainers cannot modify stats while the game sits on the main menu. Go into the game world and try to toggle the options again.

### Antivirus alerts
Some security software flags tools that interact with memory addresses as suspicious. This occurs because the trainer accesses memory reserved for the game. If your security software blocks the file, you must add the folder as an exclusion. Check your antivirus settings to allow the application to run.

### Game crashes
Close both the game and the trainer. Reopen your game first, wait for the save file to load, and then open the trainer. Using too many options at once can cause memory instability. If the game repeats this behavior, enable one feature at a time to isolate the conflict.

### Cannot find the download
If the link appears broken, check your internet connection. Ensure you download the file from the authorized source provided above. Third-party websites may host incorrect or outdated versions of this toolkit. Always verify the source URL before you run any executable file on your computer.

## 📋 Frequently Asked Questions

Do I need to install any external libraries? 
No. The toolkit contains all necessary files to run on a standard Windows installation.

Can I get banned for using this?
This tool functions offline for a single-player game. It only affects your local computer. There are no online components to report your activity.

Will this corrupt my save file?
The tool does not write data directly to your save files. It modifies the values in your computer's temporary memory while the game runs. If you close the game without saving, the changes will not remain. Save within the game if you want the modifications to persist after you exit.

Does this work on other chapters?
This version supports Chapter 5. Using it on other chapters may cause errors or crashes, so keep it limited to the supported chapter.

You can view the code on the project page. The developers maintain the project to ensure compatibility with future updates of the game. If the game updates and the trainer stops working, check the project page for a new release.
